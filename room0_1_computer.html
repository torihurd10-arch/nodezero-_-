Level 0 — Section 1: What is a Computer?

Learn

A computer is a machine that takes input, processes it, and gives output. Hardware is the physical part you can touch. Software is the instructions that tell the hardware what to do.

Job Connection

In IT support, knowing what a computer is made of helps you explain problems more clearly. If a user says the screen is blank, you can think about the monitor, cables, power, and display settings.
