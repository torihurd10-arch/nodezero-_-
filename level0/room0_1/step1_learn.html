What is a Computer?

A computer is a machine that processes information. It takes input, performs calculations, and produces output. Every computer has two main parts: hardware (the physical components you can touch) and software (the programs that tell the hardware what to do).

Hardware

CPU — the brain
RAM — short-term memory
Storage — long-term memory
Monitor — output device

Software

Operating System — controls hardware
Applications — perform tasks
Drivers — connect hardware and software
