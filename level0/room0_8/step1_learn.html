Security Fundamentals

Cybersecurity protects computers, networks, and data from attacks. Understanding these fundamentals is essential for every IT role.

🔑 Passwords

  • Use long, unique passwords for every account
  • Never reuse passwords across services
  • Use a password manager (Bitwarden, 1Password)
  • 12+ characters with mixed types

🔐 Encryption

  • Converts data to unreadable ciphertext
  • HTTPS encrypts web traffic
  • At-rest encryption protects stored data
  • End-to-end encryption: only sender/receiver can read

🎣 Phishing

  • Fake emails/sites designed to steal credentials
  • Check sender's real email address
  • Hover over links before clicking
  • Never enter passwords on unfamiliar sites

🛡️ 2FA & Firewalls

  • 2FA adds a second verification step beyond password
  • Even if your password leaks, 2FA blocks attackers
  • Firewalls filter traffic in/out of a network
  • Block unauthorized connections to your systems
