Step 1 — Learn

The Command Line Interface

The terminal (CLI) lets you control a computer by typing commands. Every IT professional and developer uses it daily. It's faster and more powerful than clicking through menus.

📂 Navigation

pwd — Print Working Directory (where am I?)

ls / dir — List files in current folder

cd foldername — Change into a folder

cd .. — Go up one folder level

📝 File Operations

mkdir name — Create a new folder

touch file.txt — Create a new file (Linux/macOS)

cp source dest — Copy a file

rm file.txt — Delete a file

🌐 Network Commands

ping google.com — Test connectivity

ipconfig / ifconfig — Show network settings

nslookup domain.com — Look up DNS

⚙️ System

cls / clear — Clear the screen

echo text — Print text to screen

cat file.txt — Show file contents

sudo — Run as administrator (Linux)
