Files, Folders, Paths & Extensions

Every piece of data on your computer is stored as a file. Files live in folders (directories), and every file has an address called a path.

📁 Files & Folders

  • Files store data: text, images, programs, audio
  • Folders organize files into groups
  • Folders can contain other folders (subfolders)
  • Every drive starts at a root: C:\ on Windows, / on Linux

🔗 Paths

  • Absolute path: Full address from root — C:\Users\Tori\resume.pdf
  • Relative path: From current location — ../Documents/resume.pdf
  • / separates folders on Linux/macOS
  • \ separates folders on Windows

📎 Extensions

  • .txt — plain text
  • .html — web page
  • .exe — executable program (Windows)
  • .pdf — portable document
  • .jpg / .png — images

🔒 Permissions

  • Read — can view the file
  • Write — can edit the file
  • Execute — can run the file as a program
  • Permissions prevent unauthorized access
