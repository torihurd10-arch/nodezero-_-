Step 1 — Learn

From Source Code to Running Program

Software begins as plain text — source code — written by humans. That code must be transformed into instructions a computer can execute.

📝 Source Code

Human-readable instructions written in a programming language like Python, JavaScript, or Java. Stored in text files (.py, .js, .java).

⚙️ Compiler vs Interpreter

Compiler (C, Java): Translates all code to machine code before running. Faster execution.
Interpreter (Python, JavaScript): Runs code line by line. Faster development.

📦 Dependencies

Most programs rely on external libraries — pre-built code that handles common tasks. Missing dependencies are a top cause of build failures.

🚀 The Build Pipeline

  1. Write source code
  2. Install dependencies
  3. Compile or run the interpreter
  4. Test the output
  5. Deploy to production
